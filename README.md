
Use the **arrows**, **[W]**, **[S]**, **[R]**, **[Space]** and **[Enter]**

**[Left arrow]** - move left
**[Right arrow]** - move right
**[Up arrow]** - move up
**[Down arrow]** - move down
**[W]** - move forward
**[S]** - move backward
**[R]** - restart
**[Space]** - the blocks start to move
**[Enter]** - just try it
